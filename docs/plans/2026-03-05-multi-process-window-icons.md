# Multi-Process Window Icons Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Show up to 3 unique process icons per window (and per session in pickers), replacing the single-icon `tmux-nerd-font-window-name` plugin with a custom script.

**Architecture:** New `tmux-window-icons.sh` script queries all panes in a window via `tmux list-panes`, deduplicates process names, maps each to an emoji icon via a bash associative array (Nix-generated), and outputs up to 3 joined icons. The reflow script and `automatic-rename-format` both call this script. Session/window pickers aggregate icons across their scope.

**Tech Stack:** Bash, Nix (flake), tmux format strings

### Task 1: Create `tmux-window-icons.sh` script

**Files:**
- Create: `scripts/tmux-window-icons.sh`

**Step 1: Write the script**

```bash
#!/usr/bin/env bash
# Returns up to 3 unique process icons for a tmux window's panes.
# Usage: tmux-window-icons <session_name> <window_index>
# Output: emoji icons joined, e.g. "🧠🐟" or "🧠❄️🐟"
#
# The icon map below is generated by Nix — edit processIcons in tmux.conf.nix, not here.

set -euo pipefail

declare -A ICON_MAP=(
@ICON_MAP@
)
FALLBACK="@FALLBACK_ICON@"
MAX_ICONS=3

session="$1"
window="$2"

# Collect unique process names preserving first-seen order
declare -A seen
declare -a unique_procs
while IFS= read -r proc; do
	[[ -z $proc ]] && continue
	if [[ -z ${seen[$proc]+x} ]]; then
		seen[$proc]=1
		unique_procs+=("$proc")
	fi
done < <(tmux list-panes -t "${session}:${window}" -F '#{pane_current_command}' 2>/dev/null)

# Map to icons, cap at MAX_ICONS
result=""
count=0
for proc in "${unique_procs[@]}"; do
	((count >= MAX_ICONS)) && break
	icon="${ICON_MAP[$proc]:-$FALLBACK}"
	result+="$icon"
	((count++))
done

echo "$result"
```

```bash
# Runs only on hook events, not every status-interval
# Pad single-width glyphs (nerd fonts, 3 UTF-8 bytes) with a trailing space
# so they match double-width emoji (4 UTF-8 bytes) for column alignment.

# Icon map (Nix-generated)
declare -A ICON_MAP=(
@ICON_MAP@
)
FALLBACK="@FALLBACK_ICON@"
MAX_ICONS=3

for ((j = 0; j < total; j++)); do
	target="${SESSION}:${indices[$j]}"

	# Collect unique processes across all panes in this window
	declare -A seen=()
	declare -a unique_procs=()
	while IFS= read -r proc; do
		[[ -z $proc ]] && continue
		if [[ -z ${seen[$proc]+x} ]]; then
			seen[$proc]=1
			unique_procs+=("$proc")
		fi
	done < <(tmux list-panes -t "$target" -F '#{pane_current_command}' 2>/dev/null)

	# Map to icons, cap at MAX_ICONS
	icon=""
	count=0
	for proc in "${unique_procs[@]}"; do
		((count >= MAX_ICONS)) && break
		icon+="${ICON_MAP[$proc]:-$FALLBACK}"
		((count++))
	done
	unset seen unique_procs

	# Nerd font glyphs are 3 bytes (PUA, 1 display col); emoji are 4 bytes (2 display cols).
	# Append a space to single-width icons for consistent 2-col alignment.
	if [[ -n $icon && ${#icon} -eq 1 ]]; then
		byte_len=$(printf '%s' "$icon" | wc -c)
		if ((byte_len <= 3)); then
			icon="$icon "
		fi
	fi
	tmux set -w -t "$target" @window_icon_display "$icon"
done
```
