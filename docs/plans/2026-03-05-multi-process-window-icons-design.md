# Multi-Process Window Icons

## Goal

Show up to 3 unique process icons per window name and in session/window pickers,
replacing the single-icon `tmux-nerd-font-window-name` plugin with a custom script.

## Requirements

- **Window names:** Show deduplicated process icons (up to 3) followed by branch/dir name.
  Example: `🧠🐟 main` instead of `🧠 main`
- **Session picker:** Show all unique process icons across the session (capped at 3).
- **Window picker:** Show multi-process icons per window.
- **Icon style:** Keep current emoji icons (🧠🐟❄️⚙️⚡).
- **Deduplication:** Unique processes only, first-seen order preserved.

## Components

### 1. `tmux-window-icons.sh` — new script

Replaces the `tmux-nerd-font-window-name` plugin.

**Interface:** `tmux-window-icons <session_name> <window_index>`
**Output:** Up to 3 emoji icons joined, e.g. `🧠🐟`

**Logic:**
1. `tmux list-panes -t "$session:$window" -F '#{pane_current_command}'`
2. Deduplicate (preserve first-seen order)
3. Map each to icon via bash associative array (generated by Nix)
4. Cap at 3 icons
5. Echo joined result

**Icon map** (Nix-generated, baked into script):
```bash
declare -A ICON_MAP=(
  [claude]="🧠" [fish]="🐟" [nh]="❄️" [nix]="❄️"
  [process-compose]="⚙️" [amp]="⚡"
)
FALLBACK=""
```

### 2. `tmux.conf.nix` changes

- Remove `tmux-nerd-font-window-name` plugin (fetchFromGitHub, run-shell, YAML config)
- Remove `nerdFontConfig` YAML generation
- Remove `XDG_CONFIG_DIRS` nerd font YAML copy in wrapper
- Move icon map from YAML to Nix `icons.processes` attrset
- Update `automatic-rename-format`:
  ```
  #(tmux-window-icons #{session_name} #{window_index}) #{?#{@branch},...}
  ```

### 3. Session picker enhancement (`tmux-session-picker.sh`)

- Before choose-tree display, iterate all windows/panes per session
- Collect unique processes across entire session
- Cap at 3 icons
- Store as `@picker_icons` session variable
- Include in choose-tree format string

### 4. Window picker enhancement (`tmux-window-picker.sh`)

- Replace single `#{window_icon}` with multi-icon output
- Pre-compute icons per window during the preparation loop
- Store in window variable for choose-tree rendering

## What gets removed

- `tmux-nerd-font-window-name` plugin (fetchFromGitHub + pinning)
- YAML config generation (`nerdFontConfig`, `yamlFormat`)
- `yq` dependency (no longer needed for icon lookup)
- `XDG_CONFIG_DIRS` override in tmux wrapper (was only for plugin's YAML)

## Performance

The script runs on every window rename (automatic-rename). The old plugin called
`yq` (parsing YAML per invocation). The new script uses a bash associative array
lookup, which is faster. `tmux list-panes` is lightweight.
